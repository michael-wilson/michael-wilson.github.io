Registration

To register for Rooting 2021 please click the 'Register Now' button above. We have 'early bird' rates available for attendees registering before 31st March 2021.

When you register you can indicate if you would like to be considered to give a talk and/or present a poster. If you choose to do this then please visit the submissions page to see the process to follow.

We have a number of bursaries to support students hoping to attend the conference, if you would like to apply for a bursary please visit this page before Registering.
